- receber um valor inteiro representando a hora (0 a 23)
- definir o horario dos turnos
- respostas
  - 1. texto turno
  - 2. menssagem de erro

- validações
  - 1. recebeu horario
  - 2. aceitar str e tentar converter se der erro retorna
  - 3. aceitar float se der erro retorna
  - 4. validar se estar no intervalo 0 a 23

- buscar qual o turno
  - 1.  
      - **05–11** → Manhã  
      - **12–17** → Tarde  
      - **18–22** → Noite  
      - **23–04** → Madrugada 
  
- retornar unico turno ou retorna unico erro
